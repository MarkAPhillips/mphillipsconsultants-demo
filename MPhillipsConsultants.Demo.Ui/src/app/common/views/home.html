Demo Application

This is a demo application built using the following technologies:

  • AngularJS, Bootstrap 3 and Gulp Javascript task runner(NodeJS) for UI
  • .NET Web API 2 using oData v4.0
  • Entity Framework 6.1.2 Code First - ORM
  • Ninject IoC, C# 4.5
  • Error logging - nLog
  • Unit tests using nUnit/Moq and Karma/Protractor for AngularJS

The purpose of this Demo is to develop an application to manage a group of users.
